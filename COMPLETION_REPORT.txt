🔧 API ENDPOINTS (8 всего)
══════════════════════════════════════════════════════════════════════════════

GET ENDPOINTS:
  GET /api/bots
    └─ Получить список всех ботов

  GET /api/bots/:botId
    └─ Получить информацию о боте

  GET /api/user/bot-subscriptions
    └─ Получить подписки пользователя

POST ENDPOINTS:
  POST /api/bots
    └─ Создать нового бота

  POST /api/bots/:botId/subscribe
    └─ Подписаться на бота

  POST /api/bots/:botId/unsubscribe
    └─ Отписаться от бота

  POST /api/bots/:botId/command
    └─ Отправить команду боту

  POST /api/bots/:botId/button-click
    └─ Обработать клик по кнопке

══════════════════════════════════════════════════════════════════════════════
💾 БАЗА ДАННЫХ (4 таблицы)
══════════════════════════════════════════════════════════════════════════════

nexis_bots
  ├── id (PRIMARY KEY)
  ├── name (VARCHAR UNIQUE)
  ├── description (TEXT)
  ├── avatar_url (TEXT)
  ├── creator_id (FK)
  ├── is_active (BOOLEAN)
  ├── commands (JSON)
  └── created_at, updated_at

nexis_bot_subscribers
  ├── id (PRIMARY KEY)
  ├── bot_id (FK)
  ├── user_id (FK)
  └── subscribed_at

nexis_bot_messages
  ├── id (PRIMARY KEY)
  ├── bot_id (FK)
  ├── chat_id (FK)
  ├── message_text (TEXT)
  ├── buttons (JSON)
  └── created_at

nexis_bot_actions
  ├── id (PRIMARY KEY)
  ├── bot_id (FK)
  ├── user_id (FK)
  ├── action_type (VARCHAR)
  ├── action_data (JSON)
  └── created_at
